Steps to run code in AWS:
1. Create an EMR cluster with Tensorflow, Hadoop and Spark
2. From the management console search for EMR 
3. Inside EMR on the left hand side select notebooks
4. Create a new notebook and assign the created cluster and click create
5. Open the notebook you have created and select "Open in JupyterLab"
6. Upload the .ipynb source file provided to the jupyter and open the notebook
7. Once you open the notebook just run all the cells to view the predictions
